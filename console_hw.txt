mkdir group_29 ; cd group_29 ; mkdir qa1 qa2 qa3 ; cd qa1 ; touch qa{1..3}.txt qa{4..5}.json ; mkdir qa4 qa5 qa6 ; ls -la ; mv {qa1,qa2}.txt qa4/

Или же создать файл-скрипт и потом его запустить
создаем файл:
vim myscript.txt
входим в режим редактирования, нажимая "i" на клавиатуре и пишем скрипт:
#!/bin/bash
mkdir group_29
cd group_29
mkdir qa1 qa2 qa3
cd qa1
touch qa{1..3}.txt qa{4..5}.json
mkdir qa4 qa5 qa6
ls -la
mv {qa1,qa2}.txt qa4/
выходим из редактора и сохраняем:
esc :wq
Сделаем файл исполняемым:
chmod +x ./myscript.txt
Запускаем скрипт:
./myscript.txt
